## 整体思路

1. 分析「京东」App 的业务逻辑结构，将整体业务按重要程度进行划分，安排整体开发流程
2. 在 Mac 环境下使用 Charles 软件抓包，抓取「京东」的网络接口数据，并整理接口
3. 使用 GetX 开发项目
